Course Description

This course is an introduction to front end web design and development using Hypertext Markup Language (HTML), Cascading Style Sheets (CSS), and various software programs. The course consists of lectures and demonstrations on the design principles necessary to create an effective website as well as how to use code and other tools to bring it to life.

Course Requirements

Throughout the semester we will be working towards developing unique, custom websites based on what we have been learning during lectures and demonstrations. There will be feedback sessions where students will present the rationale behind their work as well as working sessions to further hone the skills learned during demonstrations. This course moves quickly, and new concepts each week build on previous material. Attendance and participation is extremely important.

Course Purpose and Learning Goals

Students will learn technical skills and conceptual practices required for modern web development and how to collaborate with web developers as a designer. This foundational course will provide a basic understanding of internet history, how web development works, and the principles of good user experience and visual design on the web. By the end of the course, students should be able to build a fully functioning front-end web site.

Due to the rapid pace at which web standards and technologies evolve, the goal of this course is to give students a practical working knowledge of the latest and current design practices, as well as an understanding of the broader concepts involved. This will provide the tools and base knowledge required to remain up to date on the latest standards and practices going forward.

Grading

Grades with be determined according to the following formula:

  • 25% Final Website
  • 25% Website planning deliverables (Project brief, wireframes, design mockups)
  • 50% Weekly Assignments (including In-Class)

Course Tools

We will use several programs and tools to help build our websites:

  • Whimsical Wireframes: A tool for quick wireframing. Please make a free account. You will be added to the course's workspace when class begins.
  • GitHub.com and GitHub Desktop: GitHub is a tool for collaborating and sharing code. Please make a GitHub account and download the desktop app.
  • Figma: Figma is a design tool very popular in the industry right now. Please make an account and download the desktop app.
  • Visual Studio Code: VS Code is a popular code editing tool we will be writing our website code in. Please download the app.
