Resources List

Accessibility

  • Accessible Colors: It is crucial that the colors you use on a website have enough contrast with their background color. You can easily check this and get accessible alternatives with this tool.

Assets

  • Hero Icons: SVG icons that can be easily copied for use on a website.
  • Unsplash: Free stock photos
  • Placeholder.com: Placeholder image urls

Coding Tools

  • GitHub.com and GitHub Desktop: GitHub is a tool for collaborating and sharing code.
  • Visual Studio Code: VS Code is a popular code editing tool we will be writing our website code in.

Colors and Typography

  • Coolors: Easily generate beautiful color palettes. Make sure they are accessible.
  • Google Fonts: A large resource of free to use, beautiful and common fonts for both print and web.

Design Tools

  • Slickplan: A good tool for building site maps
  • Whimsical Wireframes: A tool for quick wireframing.
  • Figma: Figma is a design tool very popular in the industry right now.
