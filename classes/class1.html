1/24/23 - Class 1

  • Topics covered: Introductions, Course Overview, Project Briefs, Sitemaps, and Wireframing
  • Link to slides
  • Assignment:
    1. Make sure you are signed up for Whimsical, GitHub and Figma with your TCNJ email.
    2. Come up with the concept for the website project you will be building this semester and fill out the project brief (make your own copy)
    3. Design a sitemap outlining your site’s structure with Whimsical mind maps (or another tool if preferred)
    4. Design a wireframe for the homepage of your site with Whimsical wireframes (or another tool if preferred). You only have to have the homepage completed for now.

    Next week you will present all of this for feedback. Note that the goal of this assignment is not to make a perfect site map or wireframe in a certain way but rather get used to these methods for communicating design requirements, which is why I don't care if you use another tool. Design is communicating information and there is not one right way to do it, as long as it is clear.
